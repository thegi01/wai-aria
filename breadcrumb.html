Breadcrumb

A breadcrumb trail consistof a list of links to the parent pages of the current page in hierarchical order.
It helps users find their palce within a website or web application.
Breadcrumbs are often placed horizontally before a page's main content.

Keyboard Interaction

Not Applicable.

WAI-ARIA Roles, States, and Properties

  • Bread trail is contained within a navigation landmark region.
  • The landmark region is labelled via aria-label or aria-labelledby.
  • The link to the current page has aria-current set to page.
    If the element represeting the current page is not a link, aria-current is optional.
